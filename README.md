# DDRA
DDRA DNS REST API

# USAGE
* Installing npm packages
  
    `npm install`

* Without specifying port (Default port **8888** is used)

    `npm start` _**OR**_ `node index.js`
    
* Specifying port

    `npm start --port=1234` _**OR**_ `node index.js --port=1234`
