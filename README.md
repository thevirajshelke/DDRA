# DDRA
DDRA DNS REST API

# USAGE
* Installing npm packages
  
    `npm install`

* Without specifying port (Default port **8888** is used)

    `npm start` or `node index.js`
    
* Specifying port

    `npm start --port=1234` or`npm start -p 1234` or `node index.js --port=1234` or `node index.js -p 1234`
